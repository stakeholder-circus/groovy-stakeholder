- `groovy-stakeholder.live-provider-deferred`: `--experimental-provider` fails fast; no provider network/runtime adapter is enabled in tranche C.
- `groovy-stakeholder.later-family-dedicated-parity`: blockchain, quantum/runtime, healthcare, charging, MCP/A2A, streaming, and service-mesh families are grouped deterministic fallback only.
- `groovy-stakeholder.core-fixture-sync`: normalized JSON should be mirrored into `stakeholder-core` fixtures in a coordinated workspace pass, not from this repo-local task.
- `groovy-stakeholder.codeql-activation-pending`: CodeQL is parked because Groovy is not configured as a first-class analysis target here.
